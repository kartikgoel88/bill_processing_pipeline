EXPENSE TYPE: COMMUTE (cab / ride receipt — Rapido, Ola, Uber, etc.)
- amount: Use ONLY "Selected Price", "Fare", "Total" or "Price" as printed on the receipt. NEVER use Ride ID, Order ID, or any long reference number (e.g. RD173...) as amount.
- date: Use "Time of Ride" or the ride date shown; set month to YYYY-MM from that date (e.g. "Nov 14th 2024" → month "2024-11", bill_date "2024-11-14").
- vendor_name: Use the app or service name (e.g. Rapido, Ola, Uber).
